# 🎓 Student Performance Prediction System

Machine Learning web app that predicts whether a student will pass or fail based on:

- Study Hours
- Attendance
- Previous Marks

## 🚀 Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## 📊 ML Concepts
- Train/Test Split
- Logistic Regression
- Decision Tree
- Pipeline
- Model Evaluation

## ▶ Run Locally

pip install -r requirements.txt  
streamlit run app.py
